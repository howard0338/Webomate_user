About The Program

學程宗旨與推廣內容

學程以於校園中推廣創意與創業相關學習、實踐、與輔導為宗旨，期能透過多元創意發想、跨領域創業知能、務實的商業設計與包容失敗的事業實踐過程。

我們致力於激發臺大三校聯盟學生提出有效解決問題的創意，過程中學習如何面對失敗並不斷嘗試解決問題，激勵學生將創意轉化為創業行動。

  • 激發多元創意發想與跨領域合作
  • 培育務實的商業設計與創業知能
  • 鼓勵包容失敗並勇於嘗試的事業實踐

Core Pillars

核心精神與學習支柱

創意發想

打破常規框架，激發學生提出有效解決問題的創新點子，培養敏銳的觀察力與創造力。

創業知能

結合臺大三校聯盟豐富資源，提供跨領域的專業知識，建立務實且嚴謹的商業設計思維。

事業實踐

打造包容失敗的環境，讓學生在實踐過程中學習面對挫折，將創意轉化為真實的創業行動。

臺大創意創業中心致力於校園中推廣創意與創業相關學習、實踐與輔導，為學生創造專業生涯新的可能性。
